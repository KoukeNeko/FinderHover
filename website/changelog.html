9 種新檔案類型支援

HTML/網頁檔案

  • 頁面標題、Meta 描述
  • 關鍵字、作者
  • 語言設定

延伸圖片中繼資料 (IPTC/XMP)

  • 創作者、創作工具
  • 標題、描述
  • 版權資訊

Markdown 檔案

  • 標題偵測（Frontmatter 或第一個標題）
  • Frontmatter 存在指示
  • 標題/圖片/連結/程式碼區塊統計

設定檔 (JSON/YAML/TOML)

  • 鍵值數量、巢狀深度
  • 陣列數量偵測

PSD 檔案

  • 圖層數量
  • 色彩模式、位元深度
  • 解析度、透明度支援

執行檔 (Mach-O)

  • 架構偵測 (arm64/x86_64/Universal)
  • 程式碼簽署狀態
  • 最低 OS 版本、SDK 版本

App Bundle (.app)

  • Bundle ID、版本號
  • 最低 macOS 版本
  • 程式碼簽署、權限數量

SQLite 資料庫

  • 資料表/索引/觸發器/視圖統計
  • 總列數、Schema 版本
  • 使用原生 SQLite3 C API

Git 儲存庫

  • 目前分支、提交次數
  • 遠端 URL、未提交變更
  • 標籤數量

技術改進

  • 原生 SQLite3 API 提升效能
  • 所有新欄位的顯示開關設定
  • 完整三語言本地化
