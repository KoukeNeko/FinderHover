Privacy Policy

您的隱私，我們重視

FinderHover 的核心原則是本機處理。所有檔案分析都在您的電腦上完成，不會將任何檔案內容傳送到外部伺服器。

完全本機處理
無分析追蹤
無需帳號
開源可審計

最後更新：2025 年 6 月

1. 我們收集的資料

FinderHover 本身不收集、儲存或傳送任何個人資料。所有檔案中繼資料的解析都在您的電腦上本機完成，不會傳送到任何伺服器。

2. 付款處理（Paddle）

當您購買授權時，付款由 Paddle.com 處理。Paddle 會收集完成交易所需的資訊（如電子郵件地址和付款資訊）。Paddle 的隱私權政策請參閱：

paddle.com/legal/privacy

3. 授權資料

您的授權金鑰和啟用狀態儲存在本機。Paddle SDK 會定期與 Paddle 伺服器通訊以驗證授權狀態，此過程僅傳送授權金鑰和裝置識別碼，不涉及任何檔案內容或個人檔案。

4. 輔助使用 API

FinderHover 使用 macOS 輔助使用 API 來偵測 Finder 中滑鼠游標下的檔案。此功能需要您在系統設定中授予權限。輔助使用 API 僅用於取得檔案路徑，所有資料均在本機處理。

5. 分析與追蹤

FinderHover 不包含任何分析、遙測或追蹤功能。我們不收集使用資料、不追蹤使用行為、不傳送匿名統計資料。

6. 政策變更

我們可能會不時更新本隱私權政策。任何變更都會在本頁面上公佈。

7. 聯絡方式

如有任何隱私相關問題，請透過 GitHub Issues 聯繫我們： github.com/KoukeNeko/FinderHover/issues
