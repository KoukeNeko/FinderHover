選擇最適合你的安裝方式

推薦

Homebrew

使用 macOS 套件管理器安裝，自動處理 Gatekeeper 驗證，更新更方便。

brew install koukeneko/tap/finderhover

安裝步驟

  1. 開啟終端機 (Terminal)
  2. 貼上上方指令並按 Enter
  3. 等待安裝完成
  4. 在 Launchpad 中開啟 FinderHover

沒有 Homebrew？

/bin/bash -c "$(curl -fsSL https://brew.sh/install.sh)"
前往 brew.sh 了解更多

直接下載

從 GitHub Releases 下載最新版本的 .app 檔案。

下載 FinderHover.app.zip

安裝步驟

  1. 下載並解壓縮 .zip 檔案
  2. 將 FinderHover.app 拖曳到「應用程式」資料夾
  3. 首次開啟時，右鍵點擊選擇「打開」
  4. 在彈出視窗中點擊「打開」以略過 Gatekeeper
由於 App 未經 Apple 公證，需要手動允許執行

從原始碼建置

適合開發者或想要自訂功能的使用者。

git clone https://github.com/KoukeNeko/FinderHover.git

建置步驟

  1. 確保已安裝 Xcode 15 或更新版本
  2. Clone 儲存庫
  3. 執行 xcodebuild -scheme FinderHover -configuration Release
  4. 或在 Xcode 中開啟專案並按 ⌘B 建置

系統需求

macOS Sonoma 14.0+

需要 macOS Sonoma 或更新版本

輔助使用權限

需要授予輔助使用權限以偵測滑鼠位置

Apple Silicon

支援 M1/M2/M3/M4 及 M5 處理器
