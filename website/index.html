Rich file information at your fingertips.

Windows-style file previews, reimagined for macOS.

Download for macOS
brew install koukeneko/tap/finderhover
macOS 14.0+ Native Swift Accessibility API

Why FinderHover?

Designed to feel native, built to be powerful.

Smart Detection

Instant previews with adjustable delay. Intelligently auto-hides when you're renaming files or dragging items.

QuickLook Integration

See native thumbnails for PDFs, images, and documents directly in the hover window.

Privacy First

All processing happens locally on your device. Zero network access required. Built on macOS Accessibility API.

Highly Customizable

Choose between rich macOS style or minimal Windows tooltip style. Adjust size, opacity, and font scaling.

Everything You Need to Know

Support for 50+ file formats with deep metadata extraction.
