有問？有答。

為什麼需要輔助使用權限？

FinderHover 使用 macOS 輔助使用 API 來偵測你的滑鼠目前指向哪個檔案。這是唯一能夠即時追蹤 Finder 中檔案的方式，且所有資料都在本機處理，不會傳送到任何伺服器。

為什麼推薦用 Homebrew 安裝？

透過 Homebrew 安裝會自動處理 Gatekeeper 驗證，省去手動解除安全性限制的步驟。直接下載的 .app 檔案需要在「系統設定 → 隱私與安全性」中手動允許執行。

支援哪些 macOS 版本？

FinderHover 需要 macOS Sonoma 14.0 或更新版本。我們使用了 Sonoma 引入的新 API 來提供更好的效能和穩定性。

會影響系統效能嗎？

不會。FinderHover 使用 Combine 框架的 debounce 機制，只在滑鼠停留時才處理資料，CPU 使用率幾乎為零。當你不使用 Finder 時，它完全不會消耗系統資源。
