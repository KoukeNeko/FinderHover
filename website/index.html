一覽無遺，
觸手可及。

FinderHover 讓你在 Finder 中輕輕一指，
就能看見檔案的所有細節。照片的 EXIF、
影片的編碼、程式碼的行數，通通一目了然。

brew install koukeneko/tap/finderhover

或 直接下載 · 在 GitHub 上查看

適用於 macOS Sonoma 14.0 或更新版本

為 macOS 精心打造

原生 Swift 開發，整合系統 API，帶來流暢體驗。

即時預覽

可調整的懸停延遲，從 0.1 秒到 2 秒。當你重新命名或拖曳檔案時，自動隱藏。

隱私優先

所有處理都在本機進行，無需網路連線。使用 macOS 輔助使用 API，不使用 AppleScript。

高度自訂

在豐富的 macOS 風格與簡約的 Windows 提示風格之間選擇。調整大小、透明度和字體縮放。

支援超過 50 種格式

深度解析各類檔案的中繼資料，讓你不用開啟檔案就能了解一切。

有問？有答。

為什麼需要輔助使用權限？

FinderHover 使用 macOS 輔助使用 API 來偵測你的滑鼠目前指向哪個檔案。這是唯一能夠即時追蹤 Finder 中檔案的方式，且所有資料都在本機處理，不會傳送到任何伺服器。

為什麼推薦用 Homebrew 安裝？

透過 Homebrew 安裝會自動處理 Gatekeeper 驗證，省去手動解除安全性限制的步驟。直接下載的 .app 檔案需要在「系統設定 → 隱私與安全性」中手動允許執行。

支援哪些 macOS 版本？

FinderHover 需要 macOS Sonoma 14.0 或更新版本。我們使用了 Sonoma 引入的新 API 來提供更好的效能和穩定性。

會影響系統效能嗎？

不會。FinderHover 使用 Combine 框架的 debounce 機制，只在滑鼠停留時才處理資料，CPU 使用率幾乎為零。當你不使用 Finder 時，它完全不會消耗系統資源。
